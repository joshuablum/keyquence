# Keyquence
## A tool to get the keyCode (deprecated), key and code.

[![Netlify Status](https://api.netlify.com/api/v1/badges/7ba6e59c-73a6-4759-8f06-0f972a09897c/deploy-status)](https://app.netlify.com/sites/keyquence/deploys)
